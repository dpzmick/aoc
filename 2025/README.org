The elisp solutions here were mostly hand written.

Following that, I had claude code port a few of the computationally interesting problems to other languages:
- Direct port to python, using same data structures, no libraries
- Port to python using idiomatic python, numpy allowed
- Direct port to lua using same structures, no libraries
- Direct port to julia, using same structures, no libraries
- Port to julia using idiomatic julia

I then had claude also write benchmarking code and reporting code
